Questo plugin permette l'integrazione con airfoma.

== Description ==

Questo plugin aggiunge ai campi di chekout le informazioni necessarie per gestire le iscrizione degli utenti all'interno dei corsi di formazione.
Alla fine dell'ordine queste informazioni saranno automaticamente disponibili all'interno del sistema aiforma da te acquistato.

== Installation ==

Guida all'installazione


1. Installare il plugin
1. Attivare il plugin
1. Recarsi alla voce impostazioni del plugin e configurare i dati di accesso (url e token) con quelli recuperati da sistema->impostazioni all'interno di airforma

== Domande frequenti ==

= Per chi è disponibile questa integrazione? =

Questa integrazione è disponibile per tutti gli utenti che hanno l'estenzione wc abilitata all'interno del proprio piano airforma

= Come posso usare questa integrazione? =

Se vuoi usare questa integrazione ma il tuo piano non te lo permette puoi modificare il tuo piano o contattare l'assistenza tecnica che ti darà ulteriori informazioni al riguardo
